## 2. Description of the Software

AIOS-Lite is a portable, AI-augmented operating-system framework consisting of:

- A POSIX shell-based OS runtime and init system.
- The AURA (Autonomous Unified Resource Agent) AI agent and its supporting modules.
- Cross-OS bridge modules for interfacing with iOS, Android, and remote Linux systems.
- A hybrid memory subsystem (context, symbolic, and semantic layers).
- An optional LLM (Large Language Model) integration layer.

The Software is designed to run on Android (Termux), Linux desktop/server environments, and
similar POSIX-compatible platforms.
