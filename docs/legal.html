§6 Privacy Notice

Effective date: 2026-01-01

6.1 Data Collected by AIOS-Lite

AIOS-Lite is a locally running system. By default:

  • No data is transmitted to any external server by the Software itself.
  • All memory, logs, and state are stored locally inside $OS_ROOT/.
  • The AURA memory database is stored at $OS_ROOT/var/lib/aura/aura-memory.db.

6.2 LLM / AI Model

The default configuration uses a local llama.cpp model with no network connectivity. If a remote AI backend is configured, queries are subject to that third-party's privacy policy.

6.3 Bridge and Mirror Features

When using bridge features (iOS, Android, SSH), AIOS-Lite may read filesystem data from connected devices only in response to explicit User commands. Data is cached locally and never transmitted externally.

6.4 No Tracking or Analytics

AIOS-Lite contains no telemetry, analytics, crash reporting, or usage tracking.
